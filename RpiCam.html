Installation et configuration d'une caméra

L'interface Web du module RPi Cam

Ouvrir le terminal

Mise à jour du firmeware

Mettre à jour puis a niveau la distribution et tous les packages installés du Raspberry Pi :

sudo apt-get update && sudo apt-get dist-upgrade -y && sudo rpi-update

Git

Vérifier si Git (gestionnaire de version) est installé :

git --version

Sinon installer git :

sudo apt-get install git-core

Télécharger RPi Cam

Récupérez la dernière version de Rpi Cam en clonant le référentiel GitHub du projet avec :

git clone https://github.com/silvanmelchior/RPi_Cam_Web_Interface.git

Changer les droits d'accés du fichier

Avant de lancer l'installation de RpiCam.
Vous devez rendre exécutables le script pour qu'il fonctionne : 

cd RPi_Cam_Web_Interface
chmod u+x *.sh

Installation de RPi Cam

Afin lancer l'installation du Rpi Cam

Dans l'interface qui s'ouvre : RPi Cam web Interface Installer

Sélectionner install, pour apache

Par défaut le répertoire d'installation est /var/www/html, créér un sous dossier dans ce répertoire en ajoutant par exemple : RpiCam.

/var/www/html/RpiCam

Le système propose de rebooter la machine

Accèder à la configuration du Rpi 

./RPi_Cam_Web_Interface_Installer.sh

Cliquer sur 1 install mettre à jour puis à niveau, d'autre options peuvent être modifiées.
